Cho Nhà Phát Triển

Cài Đặt

Clone repository từ GitHub và cài đặt các thư viện cần thiết:

# Clone repository (thay link bằng link repo của bạn)
git clone https://github.com/AnNguyen241106/HashDuplicationImage

# Đi vào thư mục dự án
cd deduplicationimage

# Cài đặt thư viện (ví dụ với pip)
pip install -r requirements.txt

Cách Sử Dụng

Đây là một ví dụ cơ bản về cách sử dụng (ví dụ với Python):

# Import thư viện của bạn
from deduplication_image import Deduplicator

# Khởi tạo
dedup = Deduplicator()

# Tải ảnh và tìm các cụm trùng lặp
image_folder = './my-image-folder'
clusters = dedup.cluster(image_folder)

# Hiển thị kết quả
clusters.show()
